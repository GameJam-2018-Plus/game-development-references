Database Access
wiki

Here are resources related to database access in Lua.

  • DBMS-generic
    • [LuaSQL] (5.0/5.1/5.2/5.3) - ODBC, Oracle, MySQL, SQLite, JDBC and PostgreSQL (see also Dado [1] for a higher level generic API).
    • [LuaDBI] (5.1/5.2/5.3) - DB2, Oracle, MySQL, PostgreSQL, SQLite. Based on Perl's DBI, provides prepared statements, placeholders and bind parameters.
    • [Lua/APR relational database drivers] wrap APR's interface to relational database engines like SQLite, MySQL, PostgreSQL, Oracle, ODBC and FreeTDS.
    • [LuaLDAP] A Lua interface to an LDAP library [2] or ADSI [3].
    • [Lua Web Tools] (5.1/5.2) - MySQL, SQLite3, FreeTDS
    • [LuaODBC] (5.1/5.2) - ODBC bindig for Lua.

  • DBMS-specific
    • [lgdbm] (5.0/5.1) - gdbm [4].
    • [LuaBDB] (5.0) - Berkeley DB [5].
    • [luacdb] (5.1) - Dan Bernstein's cdb [6].
    • [LuaPgSQL] (5.1/5.2) - Marc Balmer's PostgreSQL binding for Lua. [7].
    • [LJSQLite3] (LuaJIT 2.0) - Pure LuaJIT library to access SQLite3 databases.
    • [lua-tinycdb] (5.1) - Michael Tokarev's tinycdb [8].
    • [luatdb] (5.1) - Andrew Tridgell's tdb [9].
    • [LuaISIS] (5.0) - some CDS/ISIS features [?].
    • [LuaCodeBase] (5.0/5.1) - Sequiter's CodeBase [10].
    • [LuaSQLite] (5.1/5.2/5.3) - SQLite 3.
    • [Lua-Sqlite3] (5.0/5.1) - Sqlite 3. Two layer architecture, multiple interfaces, coroutines, iterators and much more.
    • [Lua-wxSQLite3] (5.1) - Ulrich Telle's wxSQLite3 [11].
    • [fbclient] (5.1) - Firebird 2.0-2.5 [12], pure-Lua binding, OOP and procedural interfaces, services API, blobs, and more.
    • [redis-lua] (5.1) - a binding for Redis data store [13].
    • [tarantool-lua] (5.1/LuaJIT 2.0) - connector for Tarantool memory & persistent tuple data store [14]
    • [sidereal] (5.1) - a Lua binding for Redis, with an emphasis on non-blocking operation.
    • [oluacle (written in Japanese)] (5.2) - Oracle
    • [luasolr] (5.1) - Lua module providing http based access to [Apache Solr]
    • See also CsvUtils.
