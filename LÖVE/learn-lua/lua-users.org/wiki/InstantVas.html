Instant Vas
wiki

Instant VAS is a Software as a Service project for the Mobile VAS industry, that uses Lua for SLA monitoring.

Description: This server-side system uses a set of tools & programming languages to accomplish several of their tasks, among them: website pre-processing, SMS content servicing, back office & report generation... Lua is used to implement the automated production analysis of the log files, based on modeled business logic, which verifies the expected and observed application's behavior.

Service website: http://instantvas.com
