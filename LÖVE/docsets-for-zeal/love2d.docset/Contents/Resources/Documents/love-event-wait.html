love.event.wait

Like love.event.poll(), but blocks until there is an event in the queue.

Function

Synopsis

e, a, b, c, d = love.event.wait( )

Arguments

None.

Returns

Event e
The type of event.
Variant a
First event argument.
Variant b
Second event argument.
Variant c
Third event argument.
Variant d
Fourth event argument.
